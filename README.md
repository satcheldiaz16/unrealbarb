Integrates [https://github.com/DanielSWolf/rhubarb-lip-sync](url) into Unreal Engine 5.4.4 as an async Blueprint node. Takes as input 16k mono PCM audio data and an optional text transcript, produces as output an array of Mouth Cues. Each cue contains a mouth position (denoted by a char), a start time, and and end time. Created primarily for use in the Bushnell Virtual Mentor project for driving real time lip sync animations based on Elevenlabs TTS output.
